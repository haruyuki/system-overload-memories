This website is to introduce any new or existing players about the old Pixelmon Minecraft server, System Overload. Since its start at the end of 2015, it has gone through many names, from the original System Overload to SO II: Horizons and to the current name, netAccess Pixelmon.


The server was created by M1DN1GHTCR33P3R (Risha's name at the time), one of the regular players of a previous Pixelmon server called HeartGold Pixelmon, which shut down due to payment issues. Together with nathaniel513, another regular player of HeartGold Pixelmon, managed to run the server nicely with an average player base of around 4-5 daily.


Over the course of two years before its closing in mid-2017, there have been many amazing builds and content created on the server. On this site, you'll be able to find pictures of some of the amazing builds as well as some information about the current staff members at the time. There is also a page containing external links to still existing or archived sites related to System Overload.
